# Tic-Tac-Toe
<b>Language</b>: C<br>
<b>Compilation</b>: via Makefile<br>
<b>Options</b>: [-12h] <i>(1v1)</i>[--help, --names, --p1, --p2] <i>(AI)</i>[--ai, --help, --player]<br>
<br>
Welcome to Tic-Tac-Toe!<br>
<br>
This is a remake of the famous tic-tac-toe game in C. You can either play against an AI or in 1 versus 1.<br>
Try "./tictactoe --help" to know more about the available options.<br>
<br>
Have fun!<br>
